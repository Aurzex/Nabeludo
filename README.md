# Nabeludo – 智能课程管理 & 自动播放

> 专为西北师大继续教育平台（xbsd.lt-edu.net）设计的 Tampermonkey 用户脚本，自动扫描课程、提交学时、播放视频，让学习更高效。

---

## 功能概览

- 一键扫描课程 – 自动获取当前培训项目的全部课程包及视频列表，识别每个视频的完成状态。
- 实时进度监控 – 自动刷新每个视频的已观看时长与完成状态（已完成 / 进行中 / 未开始）。
- 自动发送学时 – 定时向服务器提交学习进度，支持断点续传；完成一个视频后自动跳转至下一个未完成视频。
- 智能发送队列 – 优先处理“进行中”的视频，避免重复提交已完成内容。
- 自动播放模式 – 在视频播放页面自动播放当前视频，结束后自动刷新并跳转至下一集。
- 悬浮操作面板 – 提供视频列表、进度条、实时日志，支持拖拽和最小化，不干扰正常浏览。
- 调试模式 – 可通过油猴菜单开启详细日志，便于排查问题。

## 快速开始

### 1. 安装用户脚本管理器
在浏览器中安装 Tampermonkey 或 Violentmonkey 扩展（从官方扩展商店获取）。

### 2. 安装脚本
- 打开 Tampermonkey 仪表盘，点击“添加新脚本”，将本项目的完整代码粘贴进去并保存。
- 或直接在浏览器中打开脚本原始文件，扩展通常会提示安装。

### 3. 登录平台
访问西北师大继续教育平台（xbsd.lt-edu.net）并登录账号。

### 4. 使用面板
登录后，页面右上角出现 Nabeludo 悬浮面板：
- 扫描课程：首次使用点击“扫描课程”，获取所有视频。
- 选择视频：在列表中点击任意未完成视频，可查看其详细进度。
- 自动发送：点击“自动发送”，脚本将依次提交所有未完成视频的学时，直至全部完成。
- 自动播放：进入视频播放页后，点击“自动播放”即可自动播放下一个视频。
- 手动控制：也可使用“播放”、“刷新”等按钮单独操作。

## 界面说明

| 区域            | 功能                                                         |
|----------------|--------------------------------------------------------------|
| 认证信息        | 显示当前 Token 和用户 ID（通常自动获取，也可手动填入）       |

| 自动播放控制    | 启动/停止自动播放，手动播放、刷新页面                       |
| 日志区域        | 实时显示脚本运行日志，支持清空                               |

## 高级功能

### 调试模式
在 Tampermonkey 菜单（脚本图标 → Nabeludo）中点击“切换调试模式”，日志区域将显示详细的调试信息，便于跟踪请求和状态变化。

### 数据持久化
脚本会自动从 localStorage 读取用户凭证（Token 和用户 ID），并在登录状态变化时同步更新，无需重复输入。

---

## 注意事项

- 本脚本仅适用于西北师大继续教育平台（xbsd.lt-edu.net），其他网站无效。
- 请保持网络稳定，发送过程中避免频繁刷新或关闭页面。
- 自动发送间隔默认 60 秒，如需调整可修改源码中 scheduleNextSend 的参数。
- 若连续失败 5 次，自动发送会停止，需手动重新启动。
- 脚本依赖 CryptoJS 库，已通过 @require 自动加载，无需额外安装。
