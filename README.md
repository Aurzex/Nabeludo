# Nabeludo - 智能课程管理 & 自动播放

> 专为西北师大继续教育平台（xbsd.lt-edu.net）设计的 Tampermonkey 用户脚本，实现课程自动扫描、视频自动播放、学时自动提交，让学习更高效。

## 简介

Nabeludo 是一个浏览器用户脚本，运行于 Tampermonkey 或 Violentmonkey 扩展之上。它能够智能扫描当前培训项目中的所有课程视频，自动提交学习进度（学时），并支持自动播放下一个视频，极大减轻重复性操作负担。

## 主要功能

- **一键扫描课程**：自动获取当前培训项目中的所有课程包，并解析出所有视频条目。
- **视频状态实时监控**：自动刷新每个视频的学习进度（已观看时长）和完成状态（已完成/进行中/未开始）。
- **自动发送学时**：定时向服务器提交当前视频的学习进度，支持断点续传；完成一个视频后自动跳转至下一个未完成的视频。
- **智能发送队列**：优先处理“进行中”的视频，避免重复发送已完成内容。
- **自动播放模式**：在视频播放页面，自动播放当前视频，并在播放完毕后自动刷新页面跳转至下一集。
- **悬浮操作面板**：提供直观的 GUI，可查看视频列表、进度条、实时日志，并支持拖拽和最小化。
- **调试模式**：通过油猴菜单可开启调试日志，便于排查问题。

## 安装与使用

### 1. 安装用户脚本管理器

首先，你需要在浏览器中安装 **Tampermonkey** 或 **Violentmonkey** 扩展（可从对应浏览器的官方扩展商店获取）。

### 2. 安装 Nabeludo 脚本

- 在 Tampermonkey 中创建新脚本，将本项目的源代码完整复制粘贴进去，保存即可。
- 或者，在浏览器中打开本脚本的原始文件，Tampermonkey 通常会自动提示安装。

### 3. 登录平台

访问西北师大继续教育平台（xbsd.lt-edu.net），正常登录账号。

### 4. 使用脚本

登录后，页面右上角会出现 **Nabeludo** 悬浮面板：

- **扫描课程**：点击“扫描课程”按钮，脚本将自动获取当前项目中的所有课程和视频列表。
- **选择视频**：在视频列表中点击任一未完成的视频，会显示该视频的详细进度。
- **自动发送**：点击“自动发送”按钮，脚本将按顺序发送所有未完成视频的学习进度，直至全部完成。
- **自动播放**：进入具体视频播放页面后，点击“自动播放”，脚本将自动播放并切换视频。
- **手动操作**：也可通过“播放”、“刷新”等按钮进行手动控制。

## 高级配置

### 调试模式

在 Tampermonkey 菜单中（脚本图标 → Nabeludo），选择 **“切换调试模式”**，可在日志中显示更详细的调试信息，有助于定位问题。

### 数据持久化

脚本会自动保存 Token 和用户 ID（从 `localStorage` 读取或手动输入），重新登录后无需再次填写。

## 注意事项

- 本脚本仅适用于西北师大继续教育平台（`xbsd.lt-edu.net`），其他网站无效。
- 请确保网络连接稳定，发送过程中避免频繁刷新页面。
- 自动发送间隔默认 60 秒，可根据需要调整源码中的 `scheduleNextSend` 参数。
- 若连续失败 5 次，自动发送将停止，需手动重试。
- 脚本依赖于 `CryptoJS` 库，已通过 `@require` 自动加载，无需额外操作。
