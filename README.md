## 课程视频时长管理工具

一个用于管理西北师范大学专业课课程视频时长并自动发送观看进度的 Tampermonkey 脚本，帮助用户高效跟踪和提交学习进度。

确保浏览器已安装 Tampermonkey 扩展（支持 Chrome、Firefox、Edge 等）。  
点击 Tampermonkey 扩展图标 →「创建新脚本」。  
清空默认内容，复制本仓库中的完整脚本代码粘贴进去。  
保存脚本（快捷键 Ctrl+S 或点击编辑器工具栏的保存按钮）。

### 使用步骤

#### 参数配置
在脚本界面中，输入以下必填信息：
- Token：用户身份验证令牌（从平台获取）。
- 用户ID：平台分配的个人用户 ID。
- 课程包ID：需学习的课程包 ID（多个 ID 用逗号分隔，如 71427846,71427837）。

#### 获取课程视频
点击「获取课程视频」按钮，脚本将自动抓取指定课程包下的所有视频信息（标题、时长、状态等）。  
视频列表会显示在「视频选择」区域，包含视频 ID、时长、学习状态等信息。

#### 发送进度
- 自动发送：点击「开始」按钮，脚本将按视频时长自动计算发送次数（每 60 秒发送一次），直至视频完成。
- 手动发送：选择单个视频后，点击「手动发送一次」可立即提交当前进度。

#### 监控状态
「操作日志」区域实时显示脚本运行状态、错误信息和进度详情。  
统计面板展示总视频数、已获取 CID 数、已完成数和发送次数。

### 功能细节
- 自动进度跟踪：根据视频总时长自动计算发送次数（每 60 秒一次），确保进度提交符合平台要求。
- 状态缓存：自动记录视频的学习状态（未学 / 已学完），跳过已完成视频。
- 日志记录：详细记录每一步操作（成功 / 失败 / 警告），便于排查问题。
- 界面交互：支持窗口最小化、拖动和关闭，操作简洁直观。

### 注意事项
1. 隐私保护：Token、用户 ID 等信息为个人敏感数据，请勿泄露给他人。
2. 合规使用：脚本仅用于辅助学习，使用时需遵守平台规则，避免滥用导致账号风险。
3. 依赖说明：脚本依赖 jQuery 和 CryptoJS 处理网络请求和数据加密，已通过 CDN 自动加载。
